# Travel Slider 🌍
https://travel-slider.harlamformula.workers.dev/

**Travel Slider** – це інтерактивний слайдер для подорожей, створений на **React** і **Vite**. Користувач може переглядати різні напрямки подорожей, перегортати слайди та розгортати опис для детальної інформації.

## Технології

- **React** – для побудови компонентів інтерфейсу  
- **Vite** – сучасний збірник для швидкого запуску і build  
- **CSS** – для стилізації слайдера  
- **JavaScript ES6+** – функціональна логіка слайдера

## Функціонал

- Перегортання слайдів кнопками «Prev» / «Next»  
- Динамічний показ опису з можливістю «Show more / Show less»  
- Адаптивний дизайн для різних екранів
