# Travel Slider (React)
Простий travel-slider, створений на React. Дозволяє переглядати картки подорожей, перемикатися між ними та розгортати довгі описи.

## Функціональність

- **Перемикання слайдів** — кнопки Prev / Next циклічно переходять між елементами.
- **Скорочений / повний опис** — кнопка “Show more / Show less” розгортає текст.
- **Відображення зображень** для кожної подорожі.
- **Модульна структура** — окремі компоненти `Slide` і `Slider`.

## Використані технології та інструменти

- **React (useState)** — керування станом та оновлення UI.
- **Масив даних (`travelData`)** — структуроване зберігання інформації про подорожі.
- **CSS-класи** — базовий стиль та адаптація під будь-який дизайн.
